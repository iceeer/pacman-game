# 🟡 Pac-Man 吃豆人游戏

一个用纯 HTML5 + Canvas 开发的经典吃豆人游戏。

## 🎮 在线游玩

👉 **[点击这里开始游戏](https://iceeer.github.io/pacman-game/)**

## 玩法说明

- **方向键 / WASD** — 控制吃豆人移动
- **Enter / 空格** — 开始 / 暂停游戏
- **手机端** — 滑动屏幕控制方向

## 游戏特性

- 🟡 经典吃豆人地图
- 👻 4 只幽灵（各有 AI 行为）
- ⚡ 大力豆 — 吃后可反击幽灵
- 🏆 最高分本地保存
- 📱 支持手机触摸操作
- ✨ 发光特效

## 技术栈

- 纯 HTML5 + Canvas + JavaScript
- 无任何依赖库
